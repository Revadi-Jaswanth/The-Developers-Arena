# Interactive Portfolio Website

## Project Overview

This project is an interactive personal portfolio website built using HTML, CSS, and JavaScript.

The website includes:
- Responsive design
- Dark mode
- Form validation
- Smooth scrolling
- Dynamic typing effect

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Features

- Responsive navigation
- Hero section
- About section
- Skills section
- Projects section
- Contact form
- Dark mode toggle
- Form validation
- Local storage support
- Typing animation
