# Personal Portfolio Website

## Project Overview

This project is a simple HTML5 personal portfolio website developed as part of Week 1 Web Development training.

The website showcases:
- About information
- Skills
- Projects
- Contact form

## Technologies Used

- HTML5

## Features

- Semantic HTML structure
- Navigation menu
- Internal page links
- Profile image
- Skills section
- Projects section
- Contact form
- External links

1. Install Visual Studio Code
2. Create project folder
3. Add index.html
4. Add images folder
