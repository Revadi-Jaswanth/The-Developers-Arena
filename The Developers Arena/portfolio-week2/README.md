# Styled Portfolio Website

## Project Overview

This project is a responsive personal portfolio website built using HTML5 and CSS3.

The project was developed as part of Week 2 Web Development training.

## Features

- Responsive navigation bar
- Hero section
- About section
- Skills section
- Projects section
- Contact form
- Modern card design
- Hover effects
- Mobile responsive design

## Technologies Used

- HTML5
- CSS3

- CSS Variables

- Hover Effects
- Responsive Design
- Media Queries
- CSS Selectors

## Setup Instructions

1. Install VS Code
2. Create project folder
3. Add index.html
4. Add style.css
5. Add images folder
6. Open index.html in browser

The website automatically adjusts for:
