# Full Stack Web Development Journey (Week 1 – Week 4)

## Project Overview

This repository contains all projects completed during the 4-week Web Development learning program.

The journey starts with basic HTML and gradually progresses into:
- CSS styling
- Responsive web design
- JavaScript interactivity
- Complete business website development

The projects demonstrate practical implementation of frontend web development concepts using:
- HTML5
- CSS3
- JavaScript

---
